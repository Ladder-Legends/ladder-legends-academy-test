# Sponsorship Logos

This folder contains logos for our community partners and sponsors.

## File Naming Convention

- Use descriptive, lowercase names with hyphens
- Preferred formats: PNG (transparent background) or SVG
- Examples:
  - `sc2pulse-logo.svg`
  - `sc4all-logo.png`
  - `safehouse-logo.png`

## Recommended Image Specifications

- **Size**: 400x400px minimum
- **Format**: PNG with transparent background or SVG
- **File size**: Keep under 200KB for optimal loading

## Adding New Sponsors

1. Upload the logo file to this directory
2. Go to the homepage
3. Click "Edit Sponsorships" button (owner only)
4. Add the new sponsor with the logo path `/sponsorships/filename.ext`
5. Save changes
